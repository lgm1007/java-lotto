# 기능 요구사항
### 로또 구매
* 구입 금액을 입력하면 구입 금액에 해당하는 만큼 로또를 발급한다.
* 로또 1장의 가격은 1000원이다.

### 지난 주 당첨 번호 비교
* 지난 주 당첨 번호와 보너스 볼 번호를 입력받는다.
* 발급된 로또와 비교하여 당첨 통계를 생성한다.
  * 3개 일치 (5000원)
  * 4개 일치 (50000원)
  * 5개 일치 (1500000원)
  * 5개 일치, 보너스 볼 일치(30000000원)
  * 6개 일치 (2000000000원)
* 총 수익률을 계산해서 출력해준다.
  * 수익률은 구매한 금액 / 총 당첨 금액 계산값을 소수점 두 자리 수까지 나타낸 수이다.
